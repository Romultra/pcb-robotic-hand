# BSc Thesis — Robotic Hand PCB Design

BSc thesis at DTU designing the first integrated PCB for a rehabilitation robotic hand that assists post-stroke patients with hand motor recovery.

The PCB replaces the existing prototype's Arduino dev-boards and loose wiring, consolidating microcontroller, motor control, storage, user feedback, and power management onto a single board.

## Repository Structure

- `plan/` — Project Definition Report (LaTeX, Overleaf submodule)
- `thesis/` — Final thesis (LaTeX, Overleaf submodule)
- `BSc Project Proposal.pdf` — Original project proposal
- `Instructions_from_an_Active_Robotic_Hand_Increase_Body_Ownership_and_Task_Clarity.pdf` — Published paper on the broader robotic hand project
